Seven visualization modes

Every shape
mathematics makes

From a basic height map to Lorenz chaos — each mode is a different lens on the same mathematical universe.

Mode 01 f(x,y)
Function Surface
z = f(x, y, t)
Plot any real-valued function of two variables. Supports all Math functions — sin, cos, exp, log, pow, abs. Animate with parameter t in real time. Adjust resolution from 16×16 up to 100×100.
Try it
Mode 02 ℂ
Complex Mapping
w = f(z), z ∈ ℂ
Visualize complex functions as 3D height maps. Domain coloring encodes argument as hue and modulus as brightness. Presets include z², z³, sin(z), exp(z), tan(z), and 1/z.
Try it
Mode 03 r(u,v)
Parametric Surface
r(u,v) ∈ ℝ³
Torus, sphere, Klein bottle, Möbius strip, helicoid, catenoid, Enneper surface, Boy's surface. Full custom x(u,v), y(u,v), z(u,v) input. Resolution up to 100×100.
Try it
Mode 04 c(t)
Space Curves
c(t) = (x, y, z)
Trefoil knot, torus knot, Lissajous 3D, figure-8 knot, Viviani curve, epicycloid, cinquefoil. Rendered as smooth 3D tubes with 1200-point precision and per-vertex color gradient.
Try it
Mode 05 F(x,y,z)
Vector Field
F : ℝ³ → ℝ³
3D arrow glyphs show direction and magnitude at each grid point. Presets: rotation, curl, sink, source, saddle, tornado. Adjustable grid density from 3³ to 10³ points.
Try it
Mode 06 ẋ=f
ODE Attractors
ẋ = f(x, y, z)
Euler-integrated trajectories of chaotic systems. Lorenz, Rössler, Duffing oscillator, Thomas cyclically symmetric. Up to 40 simultaneous trajectories with color-mapped time.
Try it
Mode 07 z²+c
Fractal Surfaces
z_{n+1} = z² + c
Escape-time iteration count mapped to a 3D height field. Mandelbrot, Burning Ship, Julia c=−0.7+0.27i, Julia c=−0.4+0.6i, Newton z³−1. Up to 200 iterations.

How it works

From equation
to geometry in zero steps

01
Choose a mode
Seven tabs across the top. f(x,y) for height maps, ℂ for complex analysis, Parametric for classic surfaces, Curves for knots, Vector Field for fluid-style viz, ODE for attractors, Fractal for escape-time geometry.
02
Type an expression
The input field evaluates JavaScript math expressions with the full Math API in scope. Type sin(sqrt(x*x+y*y)) and the surface updates as you type. History auto-saves your last 8 expressions.
03
Orbit, slice, inspect
Drag to orbit, scroll to zoom, right-drag to pan. Hit the clock icon to animate t from 0 to 2π. Hit the slicer to cut a horizontal cross-section. Click any point on the surface to inspect exact coordinates and function values.
Built for exploration

Every control
is immediate

No compile step. No export. No friction between the equation and the geometry you see.

Real-time animation
Parameter t loops 0→2π continuously at 60fps. Write expressions like sin(sqrt(x*x+y*y)+t) and watch the surface ripple live. Position updates in-place — no geometry rebuild.
Cross-section slicer
A translucent horizontal plane cuts through any surface. Drag the slider to move it. The Y coordinate at the intersection is shown live in the HUD — useful for understanding function range.
Point inspector
Click anywhere on a surface. A tooltip shows exact x, y, z coordinates and the evaluated function value at that point — computed fresh from the current expression and t.
Formula history
The last 8 expressions you've entered are saved to localStorage. They appear in the side panel as clickable history items. One tap loads any previous function instantly.
Screenshot & shareable URL
Screenshot exports a full-res PNG straight from the WebGL canvas. Share encodes mode and expression into a URL — paste it anywhere to link directly to your visualization.
Variable resolution
Each mode exposes a resolution slider. Low for fast iteration, high for publication-quality renders. f(x,y) goes from 16×16 to 100×100 — 10,201 vertices all colored by height.

Interaction reference

Controls feel
natural

Left drag
Orbit camera
Drag anywhere on the canvas to rotate around the origin. Full 360° in azimuth, 0°–180° in elevation. Auto-rotation resumes when idle.
Scroll wheel
Zoom in / out
Scroll to dolly the camera toward or away. Range 0.5–40 units. Pinch-to-zoom on mobile.
Right drag
Pan
Right-click drag to pan the camera target in XY. The look-at point tracks with the pan offset so orbiting stays centered on the object.
Click surface
Inspect point
Click any point on the rendered mesh. A tooltip shows x, y, z world coordinates, the evaluated function value f(x,y) or |f(z)|, and argument in complex mode.
⏱ Clock icon
Animate t
Toggle the time HUD. Play auto-advances t from 0 to 2π in a loop. Drag the slider to set any value manually. Expressions that reference t update immediately.
▦ Slicer icon
Cross-section
Activates a horizontal slicer plane. Drag the Y slider to move it through the scene. The slice position is shown in the coordinate HUD at lower-left.
